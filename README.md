# Learning Crystal

- simple binary clock
- /bin/puenktlich

## Install Crystal

```
apt install crystal
```

## Run

```
crystal puenktlich.cr
```

## Build

### Quick

```
crystal build puenktlich.cr
```

### Release

```
crystal build --release puenktlich.cr
```

### Static/Quick


```
crystal build --static puenktlich.cr
```

### Static/Release


```
crystal build --static --release puenktlich.cr
```
